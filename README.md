# Vako Janikashvili – Middle Backend C# Developer Portfolio

Welcome to my professional portfolio, built with a modern stack focusing on performance, clean architecture representation, and a bilingual experience (English & Georgian).

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)
- **Multilingual**: [next-intl](https://next-intl-docs.vercel.app/) (English, Georgian)
- **Email Service**: [Resend](https://resend.com/)
- **Background Effects**: [tsParticles](https://particles.js.org/)
- **SEO**: Dynamic metadata, OpenGraph, and JSON-LD for professional visibility.

---

## 🌍 Key Features

- 🔄 **Bilingual Support**: Locale-aware routing with full support for English (`/en`) and Georgian (`/ka`).
- 🎨 **Theme Toggle**: Manual switch between Light and Dark modes with persistent user preference.
- ⚡ **Performance**: Optimized production build with static generation and minimal JS overhead.
- 📱 **Responsive Design**: Tailored experience for Desktop, Tablet, and Mobile.
- 🧭 **Section Navigation**: Smooth scroll-synced section highlighting.
- 📄 **CV Download**: Direct access to my latest professional resume.

---

## 🛠️ Local Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   npm run start
   ```

---

## 🚀 Vercel Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/):

1. **Environment Variables**: Make sure to set the following in your Vercel project settings:
   - `RESEND_API_KEY`: Your API key from Resend for the contact form to work.
2. **Auto-Deployment**: Connect your GitHub repository to Vercel for automatic CI/CD on every push.
